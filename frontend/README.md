# 🎨 Frontend — Projeto Integrador

Interface web do sistema de controle de caixa desenvolvido para o Projeto Integrador.

---

## 📌 Visão Geral

O frontend é responsável pela interação do usuário com o sistema, permitindo realizar operações como abertura de caixa, registro de vendas, consulta de produtos e visualização de relatórios.

A aplicação está sendo construída com foco em simplicidade, organização e facilidade de integração com o backend.

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)

---

## 🗂️ Estrutura do Projeto

```bash
frontend/
│
├── index.html        # Página inicial
├── pages/            # Páginas internas
├── components/       # Componentes reutilizáveis
├── css/              # Estilos
└── js/               # Scripts e integração com API
```
